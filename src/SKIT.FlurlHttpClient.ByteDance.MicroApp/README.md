基于 `Flurl.Http` 的抖音小程序开放平台 HTTP API SDK。

---

### 【功能特性】

-   基于抖音小程序开放平台 API 封装。
-   提供了抖音小程序所需的 AES、MD5、SHA-1、HMAC-SHA-256 等算法工具类。
-   提供了解析回调通知事件等扩展方法。
